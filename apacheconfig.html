Installing Apache

How to install apache on to your web server

Terminal Commands

Open a terminal. Search for terminal in your apps

Update the package lists by running the command: sudo apt update
                 Install Apache by running the command: sudo apt install apache2

Verifying Installation

Open a web browser

enter "http://localhost" or "http://127.0.0.1" in the address bar. 
                If Apache is running properly, you should see the default Apache web page.

Configuring Apache

This is how to configure the server

Find and edit the configuration files

Locate the Apache configuration files

In Ubuntu or Debian, the main configuration file is located at /etc/apache2/apache2.conf.
                 Type the follwing command to edit the configuration file 
                 sudo nano /path/to/apache-config-file

Setting the configuration files and restarting apache

Type the following into the configuration files Restart Apache to apply the new configuration: sudo systemctl restart apache2 


        # Set the server's IP address and port
        Listen 80
        
        # Set the server's hostname
        ServerName 'insert website name here'
        
        # Set the directory where website files are located
        DocumentRoot 'Insert File Path Here'
        
        # Define the default file to look for when a directory is requested
